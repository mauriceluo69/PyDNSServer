# PyDNSServer
- for `.py` file
    ```powershell
    python3 dnsrelay.py
    python3 dnsrelay.py -d --addr 1.1.1.1 --filename host/dns-table.txt
    python3 dnsrelay.py -dd --addr 1.1.1.1
    ```

- run `.exe` in `cmd`
    ```cmd
    dnsrelay
    dnsrelay -d --addr 1.1.1.1 --filename host/dns-table.txt
    dnsrelay -dd --addr 1.1.1.1
    ```

- run `.exe` in `powershell`
    ```powershell
    ./dnsrelay
    ./dnsrelay -d --addr 1.1.1.1 --filename host/dns-table.txt
    ./dnsrelay -dd --addr 1.1.1.1
    ```
